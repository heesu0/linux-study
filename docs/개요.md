# 리눅스 스터디

## 스터디 목적
- 리눅스마스터 1급 자격증 취득
- 실무에 도움이 되는 리눅스 공부

<br/>

## 스터디 교재
- CentOS 7으로 리눅스마스터 1급 정복하기

<br/>

## 스터디 진행 방식
- 매주 한 소챕터씩 진행
- 발표자는 책을 읽고 전체 내용을 키워드 위주로 정리
- 참여자는 책을 읽고 본인이 중요하다고 생각한 부분을 간단히 정리

<br/>
